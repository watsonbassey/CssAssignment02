CSS, also known as Cascading Style Sheet is an important styling language used for website layout in design and used alongside HTML

Padding refers to the space between the element content and the element border.

Margin property is used to define the space around elements. It clears an area around the element.

Border property is used to set the border on an element. The CSS border properties (border-style, border-color, border-width and border-radius) are used to specify the style, color and size of the border of an element.
